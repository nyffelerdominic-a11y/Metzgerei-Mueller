## Content Guidelines

### SEO Optimization Checklist:
- [ ] Primary keyword in title
- [ ] Primary keyword in first 100 words
- [ ] 2-3 related keywords naturally integrated
- [ ] 800-1,200 word count
- [ ] Descriptive H2 and H3 subheadings
- [ ] 1-2 internal links to relevant pages
- [ ] 1 strong CTA
- [ ] Include contact information or offer
- [ ] Alt text on images
- [ ] Meta description (150-160 chars)

### Content Quality Checklist:
- [ ] Original information
- [ ] Helpful to target audience
- [ ] Accurate and factual
- [ ] Clear and well-structured
- [ ] Professional tone but conversational
- [ ] Addresses user intent
- [ ] Includes actionable tips
- [ ] No AI detection patterns (vary sentence length, use contractions, active voice)

### Image Requirements:
- At least 1-2 high-quality images
- Relevant to content
- Optimized for web (compression)
- Descriptive alt text
- Credit source if not original
- Image size: 1,200px wide minimum
