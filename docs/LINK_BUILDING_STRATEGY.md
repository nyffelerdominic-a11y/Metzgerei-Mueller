# Link Building & Outreach Strategy

**Objective:** Build domain authority and local credibility through strategic link acquisition

**Timeline:** Months 2-6 of 2026  
**Target:** 15-20 quality backlinks from relevant domains

---

## Link Building Fundamentals

### What We're Looking For
- **Relevance:** Links from food, craft, local business, or event planning sites
- **Authority:** Links from established, trusted domains (not new or spammy sites)
- **Diversity:** Links from different domains, not all from the same source
- **Anchor Text:** Varied and natural (brand name, location-based, descriptive)

### What to Avoid
- Paid link schemes (Google penalty risk)
- Directory spam
- Blog comment spam
- Irrelevant links from unrelated industries
- Links from low-quality or penalized sites

## LINK OUTREACH PROCESS

### Step 1: Research & List Building
1. Identify potential linking sites
2. Research domain authority (SEMrush, Ahrefs if available)
3. Find contact information
4. Prioritize by relevance + authority
5. Create outreach spreadsheet

### Step 2: Personalized Outreach
1. Don't use automated tools (Gmail, personal email better)
2. Personalize each pitch
3. Address by name when possible
4. Keep email concise (3-4 sentences)
5. Include clear value proposition

### Step 3: Follow-up
1. Wait 5-7 days before follow-up
2. Maximum 2-3 follow-ups
3. Accept "no" gracefully
4. Move on to next prospect

### Step 4: Link Confirmation
1. Track all link placements
2. Verify link exists (won't help if removed)
3. Monitor link health regularly
4. Watch for anchor text changes

## CRITICAL SUCCESS FACTORS
